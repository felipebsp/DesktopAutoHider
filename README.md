# DesktopAutoHider

An Application to improve your Desktop

## What is DesktopAutoHider?

DesktopAutoHider is a free application created to keep your desktop clean by hiding your desktop icons and letting them appear automatically after you move your mouse.

## Features

- 🖥️ **Hide Desktop Icons** - Keep your wallpaper clean and uncluttered
- 🖱️ **Auto-Show on Mouse Movement** - Icons appear automatically when you move your mouse
- 💻 **Lightweight & Fast** - Minimal resource usage
- ⚙️ **Easy to Use** - Simple and intuitive interface

## Requirements

Before installing the application, make sure you have:
- **.NET 8.0** or higher installed on your system

## Installation

1. Download the latest release from the [Releases](https://github.com/felipebsp/DesktopAutoHider/releases) page
2. Make sure you have .NET 8.0 installed
3. Run the application
